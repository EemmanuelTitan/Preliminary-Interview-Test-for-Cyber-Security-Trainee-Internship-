# Preliminary-Interview-Test-for-Cyber-Security-Trainee-Internship-

Cybersecurity / Network Defense Trainee Test

This repository contains my responses to a Cybersecurity and Network Defense Internship test. The test focused on practical incident response, vulnerability assessment, and current cybersecurity topics.

Overview

The test required short and precise answers (10–15 sentences each) for the following scenarios:

Phishing Email – Identifying and reporting suspicious emails.

Malware Infection – Investigating, removing, and securing affected systems.

Ransomware Attack – Immediate response, containment, and recovery planning.

Unauthorized Access Attempt – Analyzing intrusion attempts and implementing preventive measures.

Data Breach Suspicion – Verification, containment, and communication strategy for potential data leaks.

Web Application Vulnerability – Reviewing websites for vulnerabilities and using testing tools.

Network Compromise – Detecting compromised devices, isolating threats, and remediation.

Risk Management & AI Security – Assessing risks of AI systems and explaining prompt injection attacks.

Security Checklist Exercise – Key security measures for deployed applications.

Current Affairs in Cybersecurity – Notable figures in cybersecurity and technology.

Purpose

This repository demonstrates:

Knowledge of cybersecurity incident response

Practical understanding of malware, phishing, and ransomware threats

Awareness of web application and network security

Insight into AI security risks

Familiarity with prominent figures in the cybersecurity field

Usage

This repository is intended as a reference for cybersecurity learning and skill demonstration. It showcases structured approaches to real-world security scenarios that interns or junior security professionals may encounter.
